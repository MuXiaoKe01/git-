git diff 工作区与暂存区的差异

git diff HEAD 工作区与版本库之间的差异

git diff --cached 暂存区与版本库之间的差异

git diff 版本ID 版本ID 查看两个版本的差异

HEAD 表示当前版本的ID

HEAD~或者HEAD^ 表示上一个版本的ID

HAED~N 表示上n个版本的ID

git diff 版本ID 版本ID 文件名 查看两个版本的中文件的差异内容
